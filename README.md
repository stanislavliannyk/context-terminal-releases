# Context Terminal

A modern Linux terminal that remembers **command history and working directory per tab**.

Designed for developers who work with multiple projects simultaneously and want **true context isolation**.

---

## ✨ Features

- 🧠 Per-tab command history
- 📁 Per-tab working directory
- ♻️ Context preserved across restarts
- 🐧 Linux-first (AppImage)
- ⚡ Lightweight and fast
